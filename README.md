# reshub-admin
Reshubの管理画面リポジトリ


-------------------------------

## reshub-adminの初期設定

aliasの実行と環境変数のアップデート

```
$ source aliases.sh
$ admin-init
```

-------------------------------

### reshub-adminの実行コマンド

* admin画面のコンテナー起動

```
$ rhadmin
```

* admin画面のコンテナー停止

```
$ addown
```

* 実行中のコンテナーのbashに接続

```
$ adbash
```

* ログの確認

```
$ adlog
```

-------------------------------

Reshub-adminのURL及び起動ポートは
_http://localhost:8080_
